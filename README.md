# CV Multimodal Classification

This is a standalone frontend project for the Multimodal Classification assignment. 

## Overview
It visualizes the pipeline of exploring state-of-the-art vision-language models on the Flickr30k dataset. The project includes:
- Automated label generation using BERTopic and NLI.
- Zero-Shot evaluation using CLIP.
- Few-Shot evaluation using MLP and CoOp methods.

## Live Demo
Deployed via GitHub Pages.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
